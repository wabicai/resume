# 文件架构
```
src
├── base
|   ├── tbase
|   ├── tevent
|   └── tmodule
├── module
|   ├── tbusiness
|   |   ├── .....
|   ├── live
|   |   ├── .....
|   ├── statistics
|   |   ├── .....
|   └── trtc
|   |   ├── .....
|   ├──tmain
|   ├──trbac
|   ├──trequest
|   ├──tlogger
|   ├──tstate
|   ├──tsotre
|   ├──tav
|   └──tboard
```

```
| 模块名 | 功能介绍 |
| --- | --- |
| TMain | 主流程上报 |
| TEvent | TCIC事件上报，便于各组件监听，用于事件监听，如进入课堂、退出课堂 |
| TSotre | 事件消费中心、中间层。主要对获取到的原始数据进行处理，如课堂列表排序、权限通知更新 |
| TBusiness | 后台接口上报 |
| TState | 状态上报，便于各组件监听，主要用于收集课堂信息。如举手人数、成员列表 |
| event_report | 统计事件上报 |
| TBoard | 白板操作上报 |
| exception_report | 异常事件上报 | 
```
