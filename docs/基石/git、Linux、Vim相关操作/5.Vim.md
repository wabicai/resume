## Vim 主要模式

1. 正常模式（也叫命令模式）：在此模式下，你可以使用键盘上的各种命令操作文件，而不会键入对应字符。
2. 底线命令模式：允许进行指令执行。（正常模式下，键入`:`或者`/`）
3. 输入模式：允许你输入文本并进行编辑。（正常模式下，键入 i）
4. 视觉模式：允许你选择文本块以进行编辑或查看。（正常模式下，键入 v）
