# 3.内存与硬盘的存储

# 内存 RAM 和硬盘 ROM（外存）

  1. 内存速度快，比硬盘快十倍，断电后数据清空。
  2. 内存分两个区域：栈，堆
  3. 简单类型直接存在栈里，复杂类型存在堆，并在栈存放指针。
  4. localStorage 和 indexDB 存在硬盘
  5. 一般 JS，字体，图片放在内存，CSS 放在硬盘缓存。

    为什么 CSS 会放在硬盘缓存中？ 因为 CSS 文件加载一次就可渲染出来,我们不会频繁读取它,所以它不适合缓存到内存中,但是 js 之类的脚本却随时可能会执行,如果脚本在磁盘当中,我们在执行脚本的时候需要从磁盘取到内存中来,这样 IO 开销就很大了,有可能导致浏览器失去响应。
